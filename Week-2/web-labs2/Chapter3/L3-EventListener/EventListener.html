Event listeners allow for greater control over events. With event listener, multiple functions can be fired based off of a single event, by simply adding multiple event listeners to the same element.

Event listeners also allow the developer to set bubbling & capturing rules. Bubbling allows events to propagate to the parent elements of the element. Capturing allows events to propogate to the child elements of the element. Bubbling is the default behavior.
