Setting Attributes

You are already aware that JavaScript can be used to alter the InnerText of an element. Here we can demonstrate that JavaScript can also be used to modify the attributes of an element - for instance, the image URL used in an image element. This exercise also demonstrates the use of external JS files as well as an external stylesheet for the first time.

Your goal will be to write the code in the linked JS file that will change the following image to display whatever image URL is provided in the text input.
